JavaScript Fonksiyon Çağrısı

Yöntem Yeniden

Call () yöntemiyle, farklı nesnelerde kullanılabilecek bir yöntem yazabilirsiniz.

Tüm Fonksiyonlar Yöntemdir

JavaScript'te tüm işlevler nesne yöntemleridir.

Bir işlev bir JavaScript nesnesinin yöntemi değilse, genel nesnenin bir işlevidir (bkz. Önceki bölüm).

Aşağıdaki örnek, firstName, lastName, fullName olmak üzere 3 özelliğe sahip bir nesne oluşturur.

var person = {

firstName: "John",

lastName: "Doe",

fullName: function () {

return this.firstName + "" + this.lastName;

}

}

person.fullName (); // "John Doe" geri dönecek

This Anahtar Kelimesi

İşlev tanımında, this işlevin "sahibini" ifade eder.

Yukarıdaki örnekte, bu, fullName işlevine "sahip" olan kişi nesnesidir.

Başka bir deyişle, this.firstName bu nesnenin firstName özelliği anlamına gelir.

Bu anahtar kelime hakkında daha fazla bilgiyi JS bu Anahtar Kelimede bulabilirsiniz.

JavaScript call () Yöntemi

Call () yöntemi, önceden tanımlanmış bir JavaScript yöntemidir.

Bir bağımsız değişken (parametre) olarak sahip nesnesine sahip bir yöntemi çağırmak (çağırmak) için kullanılabilir.

Call () ile, bir nesne başka bir nesneye ait bir yöntemi kullanabilir.

Bu örnek, person1 üzerinde kullanarak kişinin fullName yöntemini çağırır:

var person = {

fullName: function () {

return this.firstName + "" + this.lastName;

}

}


var person1 = {

firstName: "John",

lastName: "Doe"

}


var person2 = {

firstName: "Mary",

lastName: "Doe"

}

person.fullName.call (person1); // "John Doe" geri dönecek

Bu örnek, person2 üzerinde kullanarak kişinin fullName yöntemini çağırır:

var person = {

fullName: function () {

return this.firstName + "" + this.lastName;

}

}


var person1 = {

firstName: "John",

lastName: "Doe"

}


var person2 = {

firstName: "Mary",

lastName: "Doe"

}

Bağımsız Değişkenlerle call () Yöntemi

Call () yöntemi bağımsız değişkenleri kabul edebilir:

var person = {

fullName: function (şehir, ülke) {

this.firstName + "" + this.lastName + "," + şehir + "," + ülke;

}

}

var person1 = {

firstName: "John",

lastName: "Doe"

}

person.fullName.call (person1, "Oslo", "Norveç");
