JavaScript Nesne Prototipleri

Tüm JavaScript nesneleri, özellikleri ve yöntemleri bir prototipten devralır.

Önceki bölümde, bir nesne oluşturucunun nasıl kullanılacağını öğrendik:

function Person(first, last, age, eyecolor) {

this.firstName = first;

this.lastName = last;

this.age = age;

this.eyeColor = eyecolor;

}


var myFather = new Person("John", "Doe", 50, "blue");

var myMother = new Person("Sally", "Rally", 48, "green");"

Ayrıca, mevcut bir nesne yapıcısına yeni bir özellik ekleyemeyeceğinizi de öğrendik:

Person.nationality = "English";

Bir kurucuya yeni bir özellik eklemek için, yapıcı işlevine eklemeniz gerekir:

function Person(first, last, age, eyecolor) {

this.firstName = first;

this.lastName = last;

this.age = age;

this.eyeColor = eyecolor;

this.nationality = "English";

}

Prototip Kalıtım

Tüm JavaScript nesneleri bir prototipin özelliklerini ve yöntemlerini devralır:

  • Date.prototype'den devralınan tarih nesneleri

  • Dizi nesneleri, Array.prototype'den miras alır

  • Kişi nesneleri Person.prototype miras

    • Object.prototype prototip miras zincirinin en üstünde:

    Tarih nesneleri, Dizi nesneleri ve Kişi nesneleri Object.prototype'den miras alınır.

    Nesnelere Özellik ve Yöntem Ekleme

    Bazen belirli bir türdeki mevcut tüm nesnelere yeni özellikler (veya yöntemler) eklemek istersiniz.

    Bazen bir nesne yapıcısına yeni özellikler (veya yöntemler) eklemek istersiniz.

    Prototip Özelliğini Kullanma

    JavaScript prototip özelliği, nesne oluşturuculara yeni özellikler eklemenize olanak tanır:

    function Person(first, last, age, eyecolor) {

    this.firstName = first;

    this.lastName = last;

    this.age = age;

    this.eyeColor = eyecolor;

    }


    Person.prototype.nationality = "English";

    JavaScript prototip özelliği ayrıca nesne yapıcılarına yeni yöntemler eklemenize izin verir:

    function Person(first, last, age, eyecolor) {

    this.firstName = first;

    this.lastName = last;

    this.age = age;

    this.eyeColor = eyecolor;

    }


    Person.prototype.name = function() {

    return this.firstName + " " + this.lastName;

    };

    Yalnızca kendi prototiplerinizi değiştirin. Standart JavaScript nesnelerinin prototiplerini asla değiştirmeyin.
