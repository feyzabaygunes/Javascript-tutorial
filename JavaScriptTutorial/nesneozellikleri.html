JavaScript Nesne Özellikleri

Özellikler, herhangi bir JavaScript nesnesinin en önemli parçasıdır.

JavaScript Özellikleri

Özellikler, bir JavaScript nesnesiyle ilişkili değerlerdir.

JavaScript nesnesi, sıralanmamış özelliklerin bir koleksiyonudur.

Özellikler genellikle değiştirilebilir, eklenebilir ve silinebilir, ancak bazıları salt okunurdur.

JavaScript Özelliklerine Erişme

Bir nesnenin özelliğine erişmek için sözdizimi:

objectName.property // person.age

veya

objectName ["özellik"] // kişi ["yaş"]

veya

objectName [ifade] // x = "yaş"; kullanıcı [x]

İfade bir özellik adına göre değerlendirilmelidir.

Örnek 1

person.firstname + "" + person.age + "yaşında." ;

Örnek 2

kişi ["ad"] + "" + kişi ["yaş"] + "yaşında" dir;

JavaScript for ... in döngüsü

JavaScript for ... in deyimi bir nesnenin özellikleri arasında dolaşır.

Sözdizimi

for (nesnedeki değişken) {

// yürütülecek kod

}

For ... in döngüsünün içindeki kod bloğu, her özellik için bir kez yürütülür.

Bir nesnenin özellikleri arasında döngü yapmak:

var kisi = {ad: "John", soyad: "Doe", yas: 25};

for (x in kisi) {

txt + = kisi [x];

}

Yeni Özellik Ekleme

Mevcut bir nesneye basitçe bir değer vererek yeni özellikler ekleyebilirsiniz.

Kişi nesnesinin zaten var olduğunu varsayın - daha sonra ona yeni özellikler verebilirsiniz:

kisi.nationality = "İngilizce";

Özellikleri Silme

Delete anahtar sözcüğü bir nesneden bir özelliği siler:

var kisi = {ad: "John", soyad: "Doe", yas: 50, gozRengi: "mavi"};

kisi.age dosyasını silin; // veya kişiyi sil ["age"];

Delete anahtar sözcüğü hem mülkün değerini hem de mülkün kendisini siler.

Silindikten sonra, özellik tekrar geri eklenmeden önce kullanılamaz.

Silme işleci nesne özelliklerinde kullanılmak üzere tasarlanmıştır. Değişkenler veya fonksiyonlar üzerinde hiçbir etkisi yoktur.

Delete operatörü önceden tanımlanmış JavaScript nesnesi özelliklerinde kullanılmamalıdır. Uygulamanızı kilitleyebilir.

Özellik Nitelikleri

Tüm özelliklerin bir adı vardır. Ayrıca bir de değerleri var.

Değer, mülkün özelliklerinden biridir.

Diğer özellikler şunlardır: numaralandırılabilir, yapılandırılabilir ve yazılabilir.

Bu öznitelikler özelliğe nasıl erişilebileceğini tanımlar (okunabilir mi? Yazılabilir mi?)

JavaScript'te tüm öznitelikler okunabilir, ancak yalnızca değer özniteliği değiştirilebilir (ve yalnızca özellik yazılabilirse).

(ECMAScript 5, tüm özellik niteliklerini almak ve ayarlamak için yöntemlere sahiptir)

Prototip Özellikleri

JavaScript nesneleri prototiplerinin özelliklerini devralır.

Delete anahtar sözcüğü devralınan özellikleri silmez, ancak bir prototip özelliğini silerseniz, prototipten devralınan tüm nesneleri etkiler.
