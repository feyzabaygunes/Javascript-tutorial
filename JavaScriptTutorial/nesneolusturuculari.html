JavaScript Nesne Oluşturucuları

function Kişi (ilk, son, yaş, göz) {

this.firstName = ilk;

this.lastName = son;

this.age = yaş;

this.eyeColor = göz;

}

Yapıcı işlevlerini büyük harfli ilk harfle adlandırmak iyi bir uygulama olarak kabul edilir.

Nesne Türleri (Planlar) (Sınıflar)

Önceki bölümlerden örnekler sınırlıdır.

Yalnızca tek nesneler oluştururlar.

Bazen aynı "tipte" birçok nesne oluşturmak için bir "taslak" lazım.

Bir "nesne türü" yaratmanın yolu, bir nesne yapıcı işlevi kullanmaktır.

Yukarıdaki örnekte, Person () işlevi bir nesne yapıcı işlevidir.

New anahtar kelimesiyle yapıcı işlevi çağrılarak aynı türdeki nesneler oluşturulur:

var myFather = yeni Kişi ("John", "Doe", 50, "mavi");

var myMother = yeni Kişi ("Sally", "Ralli", 48, "yeşil");

This Anahtar Kelimesi

JavaScript'te buna denen şey, koda "sahip" olan nesnedir.

This in değeri, bir nesnede kullanıldığında, nesnenin kendisidir.

Bir yapıcı işlevinde bunun bir değeri yoktur. Yeni nesnenin yerine geçer. Yeni bir nesne oluşturulduğunda bunun değeri yeni nesne olacaktır.

This'in bir değişken olmadığını unutmayın. Bir anahtar kelime. Bunun değerini değiştiremezsiniz.

Bir Nesneye Özellik Ekleme

Mevcut bir nesneye yeni bir özellik eklemek kolaydır:

myFather.nationality = "İngilizce";

Özellik myFather'a eklenecek. Anneme değil. (Başka hiçbir nesneye değil).

Bir Nesneye Yöntem Ekleme

Mevcut bir nesneye yeni bir yöntem eklemek kolaydır:

myFather.name = function () {

return this.firstName + " " + this.lastName;

};

Yöntem myFather'a eklenecektir. Anneme değil. (Başka hiçbir nesneye değil).

Yapıcıya Özellik Ekleme

Varolan bir nesneye yeni bir özellik eklediğiniz gibi, nesne oluşturucuya yeni bir özellik ekleyemezsiniz:

Person.nationality = "İngilizce";

Bir kurucuya yeni bir özellik eklemek için, kurucuyu yapıcı işlevine eklemeniz gerekir:

function kisi (ilk, son, yaş, göz rengi) {

this.firstName = ilk;

this.lastName = son;

this.age = yaş;

this.eyeColor = göz rengi;

this.nationality = "İngilizce";

}

Bu şekilde nesne özellikleri varsayılan değerlere sahip olabilir.

Bir Yapıcıya Yöntem Ekleme

Yapıcı işleviniz ayrıca yöntemleri de tanımlayabilir:

funciton kisi (ilk, son, yaş, göz rengi) {

this.firstName = ilk;

this.lastName = son;

this.age = yaş;

this.eyeColor = göz rengi;

this.name = function() {return this.firstName + " " + this.lastName;};

}

Varolan bir nesneye yeni bir yöntem eklediğiniz gibi, nesne oluşturucuya yeni bir yöntem ekleyemezsiniz.

Nesne yapıcısına yöntem ekleme, yapıcı işlevi içinde yapılmalıdır:

function Person(firstName, lastName, age, eyeColor) {

this.firstName = firstName;

this.lastName = lastName;

this.age = yaş;

this.eyeColor = eyeColor;

this.changeName = function (name) {

this.lastName = name;

};

}

ChangeName () işlevi, adın değerini kişinin lastName özelliğine atar.

Şimdi deneyebilirsiniz:


myMother.changeName ( "Doe");

JavaScript, bunu myMother ile "değiştirerek" hangi kişiden bahsettiğinizi bilir.

Yerleşik JavaScript Oluşturucuları

JavaScript yerel nesneler için yerleşik yapıcılara sahiptir:

var x1 = new Object(); // A new Object object

var x2 = new String(); // A new String object

var x3 = new Number(); // A new Number object

var x4 = new Boolean(); // A new Boolean object

var x5 = new Array(); // A new Array object

var x6 = new RegExp(); // A new RegExp object

var x7 = new Function(); // A new Function object

var x8 = new Date(); // A new Date object

Math () nesnesi listede değil. Math küresel bir nesnedir.

new anahtar kelimesi Math'te kullanılamaz.

Biliyor musun?

Yukarıda görebileceğiniz gibi JavaScript'in ilkel veri türleri olan String, Number ve Boolean nesne sürümleri vardır. Ancak karmaşık nesneler oluşturmak için hiçbir neden yoktur. İlkel değerler çok daha hızlıdır.

AYRICA:

new Object () yerine {} nesne değişmezlerini kullanın.

Yeni String () yerine "" dize değişmezlerini kullanın.

new Number () yerine 12345 sayı değişmezlerini kullanın.

new Boolean () yerine true / false boolean değişmezlerini kullanın.

new Array () yerine dizi değişmezlerini [] kullanın.

new RegExp () yerine kalıp değişmezlerini / () / kullanın.

new Function () yerine işlev ifadelerini () {} kullanın.

var x1 = {}; // yeni nesne

var x2 = ""; // yeni ilkel dize

var x3 = 0; // yeni ilkel sayı

var x4 = yanlış; // yeni ilkel boole

var x5 = []; // yeni dizi nesnesi

var x6 = / () / // yeni normal ifade nesnesi

var x7 = function () {}; // yeni işlev nesnesi

String Nesneleri

Normalde, dizeler ilkel olarak oluşturulur: var firstName = "John"

Ancak dizeler, yeni anahtar sözcük kullanılarak nesneler olarak da oluşturulabilir: var firstName = new String ("John")

Dizelerin neden JS Dizeleri bölümünde nesne olarak oluşturulmaması gerektiğini öğrenin.

Number Nesnesi

Normalde, sayılar ilkel olarak oluşturulur: var x = 123

Ancak sayılar, yeni anahtar sözcük kullanılarak nesneler olarak da oluşturulabilir: var x = new Number (123)

JS Numaraları bölümünde sayıların neden nesne olarak oluşturulmaması gerektiğini öğrenin.

Boolean Nesneleri

Normal olarak, booleans ilkel olarak oluşturulur: var x = false

Ancak booleans yeni anahtar kelime kullanılarak nesne olarak da oluşturulabilir: var x = new Boolean (false)

Booleanların neden JS Booleans bölümünde nesne olarak oluşturulmaması gerektiğini öğrenin.
