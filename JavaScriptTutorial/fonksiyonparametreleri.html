JavaScript Fonksiyon Parametreleri

JavaScript işlevi, parametre değerleri (bağımsız değişkenler) üzerinde herhangi bir denetim gerçekleştirmez.

İşlev Parametreleri ve Bağımsız Değişkenler

Bu öğreticinin başlarında, işlevlerin parametreleri olabileceğini öğrendiniz:

function functionName (parametre1, parametre2, parametre3) {

// yürütülecek kod

}

İşlev parametreleri, işlev tanımında listelenen adlardır.

İşlev bağımsız değişkenleri, işleve iletilen (ve bu işlev tarafından alınan) gerçek değerlerdir.

Parametre Kuralları

JavaScript işlev tanımları, parametreler için veri türlerini belirtmez.

JavaScript işlevleri, iletilen bağımsız değişkenlerde tür denetimi gerçekleştirmez.

JavaScript işlevleri, alınan bağımsız değişkenlerin sayısını kontrol etmez.

Parametre Varsayılanları

Bir işlev eksik argümanlarla (bildirilenden daha az) çağrılırsa, eksik değerler şu şekilde ayarlanır: tanımsız

Bazen bu kabul edilebilir, ancak bazen parametreye varsayılan bir değer atamak daha iyidir:

function myFunction (x, y) {

if (y === tanımsız) {

y = 0;

}

}

ECMAScript 2015, işlev bildiriminde varsayılan parametre değerlerine izin verir:

function (a = 1, b = 1) {

// fonksiyon kodu

}

Bağımsız Değişkenler Nesnesi

JavaScript işlevlerinde arguments nesnesi adı verilen yerleşik bir nesne vardır.

Bağımsız değişken nesnesi, işlev çağrıldığında (çağrıldığında) kullanılan bir dizi bağımsız değişken içerir.

Bu şekilde, bir sayı listesindeki en yüksek değeri bulmak (örneğin) için bir işlevi kullanabilirsiniz:

Bir işlev çok fazla bağımsız değişkenle (bildirilenden fazla) çağrılırsa, bu bağımsız değişkenlere arguments nesnesi kullanılarak erişilebilir.

Bağımsız Değişkenler Değerden Geçirilir

İşlev çağrısındaki parametreler, işlevin bağımsız değişkenleridir.

JavaScript bağımsız değişkenleri değere göre iletilir: İşlev, bağımsız değişkenin konumlarını değil, yalnızca değerleri tanır.

Bir işlev bir bağımsız değişkenin değerini değiştirirse, parametrenin orijinal değerini değiştirmez.

Değişkenlerde yapılan değişiklikler işlevin dışında görünmez (yansıtılmaz).

Nesneler Referans ile Geçirilir

JavaScript'te nesne başvuruları değerlerdir.

Bu nedenle, nesneler referans olarak iletildikleri gibi davranacaktır:

Bir işlev bir nesne özelliğini değiştirirse, orijinal değeri değiştirir.

Nesne özelliklerinde yapılan değişiklikler işlevin dışında görünür (yansıtılır).
