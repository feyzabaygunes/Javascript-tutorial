Görüntüleme Nesneleri

JavaScript Nesneleri Nasıl Görüntülenir?

Bir JavaScript nesnesinin görüntülenmesi [object Object] çıktısını verecektir.

var kisi = {isim: "John", yas: 30, sehir: "New York"};

document.getElementById ("demo"). innerHTML = kisi;

JavaScript nesnelerini görüntülemek için bazı yaygın çözümler şunlardır:

  • Nesne özelliklerini ada göre görüntüleme

  • Bir döngüde nesne özelliklerini görüntüleme

  • Object.values() kullanarak object'i görüntüleme

  • Nesneyi JSON.stringify () kullanarak görüntüleme

    • Nesne Özelliklerini Görüntüleme

    Bir nesnenin özellikleri dize olarak görüntülenebilir:

    var kisi = {ad: "John", yas: 30, sehir: "New York"};

    document.getElementById ("demo"). innerHTML = person.ad + ",

    "+ person.yas +", "+ person.sehir;

    Nesneyi Bir Döngüde Görüntüleme

    Bir nesnenin özellikleri bir döngü içinde toplanabilir:

    var x, txt = "";

    var kisi = {ad: "John", yas: 30, sehir: "New York"};

    for (x in kisi) {

    txt + = kisi [x] + "";

    };

    document.getElementById ("demo"). innerHTML = txt;

    Döngüde kisi [x] kullanmalısınız.

    person.x çalışmaz (çünkü x değişkendir).

    Object.values() kullanma

    Herhangi bir JavaScript nesnesi Object.values ​​() kullanılarak bir diziye dönüştürülebilir:

    var kisi = {ad: "John", yas: 30, sehir: "New York"};


    var myArray = Object.values ​​(person);

    myArray artık görüntülenmeye hazır bir JavaScript dizisidir:

    var kisi = {ad: "John", yas: 50, sehir: "New York"};


    var myArray = Object.values ​​(kisi);

    document.getElementById ("demo"). innerHTML = myArray;

    Object.values ​​() 2016'dan beri tüm büyük tarayıcılarda desteklenmektedir.

    JSON.stringify () kullanma

    Herhangi bir JavaScript nesnesi, JSON.stringify () JavaScript işleviyle dizilebilir (bir dizeye dönüştürülebilir):

    var kisi = {ad: "John", yas: 30, sehir: "New York"};


    var myString = JSON.stringify (kisi);

    myString artık görüntülenmeye hazır bir JavaScript dizesidir:

    var kisi = {isim: "John", yas: 30, sehir: "New York"};


    var myString = JSON.stringify (kisi);

    document.getElementById ("demo"). innerHTML = myString;

    Sonuç JSON gösterimini izleyen bir dize olacaktır:


    {"ad": "John", "yaş": 50, "şehir": "New York"}

    JSON.stringify (), JavaScript'e dahildir ve tüm büyük tarayıcılarda desteklenir.

    Dizgi Tarihleri

    JSON.stringify tarihleri dizelere dönüştürür:

    var kisi = {ad: "John", bugün: new Date ()};


    var myString = JSON.stringify (kisi);

    document.getElementById ("demo"). innerHTML = myString;

    Stringify İşlevleri

    JSON.stringify işlevleri dize etmez:

    var kisi = {ad: "John", yas: function () {return 30;}};

    var myString = JSON.stringify (kisi);

    document.getElementById ("demo"). innerHTML = myString;

    Dizeleri bağlamadan önce işlevleri dizelere dönüştürürseniz bu "sabit" olabilir.

    var kisi = {ad: "John", yas: function () {return 30;}};

    person.yas = person.yas.toString ();


    var myString = JSON.stringify (kisi);

    document.getElementById ("demo"). innerHTML = myString;

    Stringify Dizileri

    JavaScript dizilerini dizgi yapmak da mümkündür:

    var arr = ["John", "Peter", "Sally", "Jane"];

    var myString = JSON.stringify (arr);

    document.getElementById ("demo"). innerHTML = myString;

    Sonuç JSON gösterimini izleyen bir dize olacaktır:

    [ "John", "Peter", "Sally", "Jane"]
