JavaScript Nesne Erişimcileri

JavaScript Erişicileri (Getter ve Setter)

ECMAScript 5 (2009) Getter ve Setters'ı tanıttı.

Harfler ve ayarlayıcılar Nesne Erişimcilerini (Hesaplanan Özellikler) tanımlamanızı sağlar.

JavaScript Getter (Get Anahtar Kelimesi)

Bu örnek, language özelliğinin değerini almak için bir lang özelliği kullanır.

// Nesne oluşturma:

var kisi = {

ad: "John",

soyadi: "Doe",

dil: "en",

get dil () {

return dil;

}

};


// Alıcıyı kullanarak nesneden veri görüntüleme:

document.getElementById ("demo"). innerHTML = kisi.dil;

JavaScript Setter (Set Anahtar Kelimesi)

Bu örnek, language özelliğinin değerini ayarlamak için bir lang özelliği kullanır.

var kisi = {

adi: "John",

soyadi: "Doe",

dil: "",

set dil (dil) {

this.language = dil;

}

};


// Bir ayarlayıcı kullanarak bir nesne özelliği ayarlama:

kisi.dil = "en";


// Nesneden veri görüntüleme:

document.getElementById ( "demo"). innerHTML = person.language;

JavaScript İşlevi veya Getter?

Bu iki örnek arasındaki farklar nelerdir?

Örnek 1

var kisi = {

adi: "John",

soyadi: "Doe",

tamAdi: function () {

return this.adi + "" + this.soyadi;

}

};


// Bir yöntem kullanarak nesneden veri görüntüleme:

document.getElementById ("demo"). innerHTML = kisi.tamAdi ();

Örnek 2

var kisi = {

adi: "John",

soyadi: "Doe",

tamAdi () {

return this.adi + "" + this.soyadi;

}

};


// Alıcıyı kullanarak nesneden veri görüntüleme:

document.getElementById ("demo"). innerHTML = person.fullName;

Örnek 1 tamAdi işlevine erişim: kisi.tamAdi ().

Örnek 2 tamAdi'na özellik olarak erişin: kisi.tamAdi.

İkinci örnek daha basit bir sözdizimi sağlar.

Veri kalitesi

JavaScript, alıcıları ve ayarlayıcıları kullanırken daha iyi veri kalitesi sağlayabilir.

Bu örnekte lang özelliğini kullanma, language özelliğinin değerini büyük harf olarak döndürür:

// Nesne oluşturma:

var kisi = {

adi: "John",

soyadi: "Doe",

dil: "en",

get dil () {

return this.language.toUpperCase();

}

};


// Alıcıyı kullanarak nesneden veri görüntüleme:

document.getElementById ("demo"). innerHTML = person.lang;

Bu örnekte lang özelliğini kullanmak, language özelliğinde bir büyük harf değeri depolar:

var kisi = {

adi: "John",

soyadi: "Doe",

dil: "",

set dil (dil) {

this.language = dil.toUpperCase ();

}

};


// Bir ayarlayıcı kullanarak bir nesne özelliği ayarlama:

kisi.dil = "en";


// Nesneden veri görüntüleme:

document.getElementById ("demo"). innerHTML = kisi.language;

Neden Harf ve Ayarlayıcılar Kullanılmalı?

  • Daha basit bir sözdizimi sağlar

  • Özellikler ve yöntemler için eşit sözdizimine izin verir

  • Daha iyi veri kalitesi sağlayabilir

  • Sahne arkasındaki işleri yapmak için kullanışlıdır

    • Object.defineProperty ()

    Getters ve Setters eklemek için defineProperty () yöntemi de kullanılabilir:

    // Nesneyi tanımla

    var obj = {sayaç: 0};


    // Ayarlayıcıları tanımla

    Object.defineProperty (obj, "sıfırla", {

    get: function () {this.counter = 0;}

    });


    Object.defineProperty (obj, "artış", {

    get: function () {this.counter ++;}

    });


    Object.defineProperty (obj, "azalma", {

    get: function () {this.counter--;}

    });


    Object.defineProperty (obj, "ekle", {

    set: function (değer) {this.counter + = değer;}

    });


    Object.defineProperty (obj, "çıkarma", {

    set: function (değer) {this.counter - = değer;}

    });


    // Sayaçla oynayın:

    obj.reset;

    obj.add = 5;

    obj.subtract = 1;

    obj.increment;

    obj.decrement;

    Tarayıcı Desteği

    Getters ve Setters, Internet Explorer 8 veya önceki sürümlerde desteklenmez.
