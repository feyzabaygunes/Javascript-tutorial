JavaScript Apply Fonksiyonu

Yöntem Yeniden

Apply () yöntemiyle, farklı nesnelerde kullanılabilecek bir yöntem yazabilirsiniz.

JavaScript Apply () Yöntemi

Apply () yöntemi, call () yöntemine (önceki bölüm) benzer.

Bu örnekte, kişinin fullName yöntemi person1'e uygulanır:

var person = {

fullName: function () {

return this.firstName + "" + this.lastName;

}

}

var person1 = {

name: "Mary",

lastName: "Doe"

}

person.fullName.apply (person1); // "Mary Doe" geri dönecek

Call () ve Apply () Arasındaki Fark

Call () yöntemi bağımsız değişkenleri ayrı ayrı alır.

Apply () yöntemi, bağımsız değişkenleri dizi olarak alır.

Bağımsız değişken listesi yerine bir dizi kullanmak istiyorsanız, Apply () yöntemi çok kullanışlıdır.

Bağımsız Değişkenlere Apply () Yöntemi

Apply () yöntemi, bir dizideki bağımsız değişkenleri kabul eder:

var person = {

fullName: işlev (şehir, ülke) {

this.firstName + "" + this.lastName + "," + şehir + " , "+ ülke;

}

}

var person1 = {


firstName: "John",

lastName: "Doe"

}

person.fullName.apply (kişi1, ["Oslo", "Norveç"]);

Call () yöntemiyle karşılaştırıldığında:

var person = {

fullName: işlev (şehir, ülke) {

this.firstName + "" + this.lastName + "," + şehir + "," + ülke;

}

}


var person1 = {

firstName: "John",

lastName: "Doe"

}

person.fullName.call (kişi1, "Oslo", "Norveç");

Dizilerde Max Yöntemi Benzetme

Math.max () yöntemini kullanarak en büyük sayıyı (sayı listesinden) bulabilirsiniz:

Math.max(1,2,3); // 3 döndürür

JavaScript dizilerinin bir max () yöntemi olmadığından, bunun yerine Math.max () yöntemini uygulayabilirsiniz.

Math.max.apply(null, [1,2,3]); // Ayrıca 3 döndürür

İlk argüman (null) önemli değil. Bu örnekte kullanılmamaktadır.

Bu örnekler aynı sonucu verecektir:

Math.max.apply(Math, [1,2,3]); // Ayrıca 3 döndürür

Math.max.apply(" ", [1,2,3]); // Ayrıca 3 döndürür

Math.max.apply(0, [1,2,3]); // Ayrıca 3 döndürür

JavaScript Sıkı Modu

JavaScript katı modunda, Apply () yönteminin ilk argümanı bir nesne değilse, çağrılan işlevin sahibi (nesne) olur. "Katı olmayan" modda, küresel nesne haline gelir.
